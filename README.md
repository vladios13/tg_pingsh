# Telegram PINGSH
Небольшой bash-скрипт для пинга сервера/ip-адреса, с отправкой уведомления в Telegram.

Все настройки проекта производятся в файле - ping.sh. 

------------
 
## Запуск по Cron  
Скрипт выполняется каждую минуту: 

```bash
  */1  *  * * *   root    /home/ping.sh
```
